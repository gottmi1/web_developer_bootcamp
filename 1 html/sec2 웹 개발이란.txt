웹

월드 와이드 웹 = 인터넷을 통해 문서나 리소스를 공유할 수 있는 정보 시스템
이러한 문서나 리소스는 URL, 즉 균일 리소스 로케이터로 식별됨.

웹 서버 = 웹을 통해 들어오는 요청을 충족시키는 역할
클라이언트 = 요청하는 쪽
서버 = 요청 받은 걸 주는 쪽

요청 -> 무엇에 대한 지침인지에 대한 응답을 받고(소스?) -> (html,css,js로 받아옴) -> 맞는 지침을 받아 웹 페이지를 구축(렌더링)

스택 
프론트 엔드 = 서빙 , 클라이언트가 요청 -> 백엔드가 돌려준 걸 클라이언트에게 다시 보여줌
백엔드 = 주방 ,요청을 받고 그에 맞는 것을 도출해 html css js로 만든 후 돌려줌 이 돌려받는 것을 보기좋게 만드는 게 프론트 엔드가 하는 일

HTML = 페이지의 내용, CSS로 꾸미지 않으면 못생겼지만 기반이 되는 구조.

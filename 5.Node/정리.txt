정적 웹사이트 = 페이지를 열 때마다 항상 같은 파일과 스크립트가 보이는 것

npm init 으로 json파일을 만들 수 있다
json파일의 "dependencies" 객체에는 현재 인스톨한 라이브러리들이 들어있다.(아무것도 인스톨 하지 않았으면 비어있음)
라이브러리는 index.js에서
const abc = require("라이브러리 명");
으로 불러와야 함. 사용 법은 라이브러리마다 다르기 때문에 npm을 참조해야 한다.

npm upgrade로 최신 버전으로 업데이트 가능.

라우팅이란? 
들어오는 요청이나 요청된 경로를 알아내고 응답의 일부 코드와 매치시키는 것

nodemon을 사용해서 코드의 내용이 바뀔 때 마다 node를 수동으로 재시작 할 필요 없이 변경을 감지할 때마다 자동 재시작되게 할 수 있다.
설치 : npm i -g nodemon
